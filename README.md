# Duolingo-ai-layoffs

## Overview
This project aims to analyze the reation of the general public to the AI-driven Duolingo layoffs of late 2023/early 2024
Comments are classified using Groq API to compare public perception with company narrative

## Features
1. Automatically fetch comments from youtube using keywords
2. Classify comments into cathegories:
-Fear of precarity
-Ethical concerns
-Quality concerns
3. Identify tone:
   -Critical
   -Neutral
   -Supportive
4. Save raw and classified data into CSV files
5. Generate visualisations of public sentiment
